Migration Guide

This document lists changes in the engine to help you to migrate your code base.

0.11.0

  • CameraMode is renamed to ProjectionMode. The corresponding parameter also changed in EmbeddedViewer from cameraMode to projectionMode.

0.8.21

  • Init3DViewerElements must be called when the document is loaded and returns an array of the created EmbeddedViewer objects. Earlier it registered a load event handler inside, now it's the responsibility of the caller.

0.8.20

  • Init3DViewerElement has been renamed to Init3DViewerFromUrlList.
  • Init3DViewerFromFileList is now available to initialize the viewer based on File objects.
